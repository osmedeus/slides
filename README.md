# Osmedeus — A Modern Orchestration Engine for Security

Slide deck for presenting [Osmedeus](https://github.com/j3ssie/osmedeus), built with [Slidev](https://sli.dev/).

## Prerequisites

- [Bun](https://bun.sh/) >= 1.0

## Setup

```bash
bun install
```

## Development

Start the slide deck in dev mode with hot reload:

```bash
bun run dev
```

## Build

Build the slides for production (static HTML):

```bash
bun run build
```

## Export to PDF

```bash
bun run export
```
